
# Customer Support Chatbot (AI/ML Project)

## Overview
This is a professional AI-powered Customer Support Chatbot designed for modern businesses.
It supports English and Roman Urdu, handles FAQs, order issues, refunds, and escalations.

## Features
- NLP-based intent handling
- Roman Urdu + English support
- Human escalation flow
- Secure & privacy-safe
- Ready for FYP / Portfolio / GitHub

## Tech Stack
- Python
- Flask
- Transformers (LLM-ready)
- HTML/CSS (basic UI)

## How to Run
```bash
pip install -r requirements.txt
python app.py
```
